# Game Introduction

開啟之後會有遊戲選單，有踩地雷、貪吃蛇等遊戲可進入遊玩。
每次進入關卡，通關條件固定、畫面狀態隨機，能留下過往最高紀錄，遊戲有暫停鍵可供玩家暫停遊戲與跳出。
贏了會得到遊戲幣，遊戲幣可以買解鎖不同頭貼或介面。
